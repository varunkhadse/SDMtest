Write a spring boot REST applicaiton which will provide Create, Get and delete operations for Product resource.

Product has the followin attributes:

a. ID(unique identifier for each product, auto generated by JPA)
b. Name
c. Code (Uniquie string for each product)
d. Date of manufacturing
e. Category (should be an Enum, example: Fashion, Electronics)

-> You can use any relational databse.
-> Resource created usin Create endpoint should be retrieved usin Get.
-> Response codes should be as per REST guidelines.
-> Error handling in case of failures.
-> Unit test for Data layer for getting a product.
